CakePHP asset versioning

2011-01-12

Yet another undocumented feature in CakePHP - very useful if you’re looking for it, that auto-versions images, css files and javascript files to force cache invalidation by browsers.

Just add:

Configure::write('Asset.timestamp', true);

to timestamp files only if debug if set to greater than 0.

Or:

Configure::write('Asset.timestamp', 'force');

to timestamp files even if debug is set to zero.

That adds a ?<LAST_MODIFIED_DATE> to the url for images, javascript files and css files, forcing people’s browsers to invalidate their cache when the files have changed.
